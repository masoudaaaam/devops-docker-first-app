# DevOps Docker First App

This is my first practical Docker project.  
The goal of this project is to learn how to containerize a simple static web application using Docker and Nginx.

## Project Overview

This project contains a simple HTML web page that runs inside a Docker container.

The application is served by Nginx using a custom Docker image.

## Technologies Used

- HTML
- Docker
- Nginx
- Git
- GitHub
